# Security Policy

## Supported Versions

This project is under active development. Security updates are provided on the latest `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities by email:

- Email: knguyen1411b@gmail.com
- Subject: `[Security] link-shortener vulnerability report`

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

## Response Process

- Initial response target: within 72 hours
- Status updates are shared until resolution
- Confirmed vulnerabilities are fixed and disclosed responsibly
